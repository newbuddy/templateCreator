How to use?

The left side of the page shows a sample template of html. Alter this page according to your requirements and get the code. Click on any element of the sample template to select it. After selecting an element a sliding div with attributes will show up. Click on the attribute you want to change and make the changings. To remove the selected element click on the remove button. An element once removed cannot be restored so be sure of your decision. You can also edit the text of selected element by using the keyboard keys to alter the text. Use backspace to delete a character. After finalizing the complete style of the sample template click on the getcode button to get the html code of the finalized template. Clicking on the getcode button will slide another div with a textarea containing the html code. Use back button if you need to go back to the attributes. Select the width in px of your webpage and copy the code shown in the textarea. Make an html file and paste the copied code in the file. Save the file and run it in the browser to show the html page.
